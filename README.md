# Bonneville Power Administration

The Bonneville Power Administration (BPA) is a federal agency within the U.S. Department of Energy that markets wholesale electrical power from federal hydroelectric projects in the Pacific Northwest. BPA also operates and maintains about three-quarters of the high-voltage transmission in the Pacific Northwest. The agency provides publicly available GIS data, energy statistics, and operational data through its data hub and web services.

**URL:** [https://www.bpa.gov](https://www.bpa.gov)

**APIs.yml:** [View APIs Index](https://raw.githubusercontent.com/api-evangelist/bonneville-power-administration/refs/heads/main/apis.yml)

## APIs and Data Services

### BPA GIS Data Hub API
The BPA GIS Data Hub at [data-bpagis.hub.arcgis.com](https://data-bpagis.hub.arcgis.com) provides publicly available geospatial data built on the ArcGIS platform. Supported formats and API types include:
- **GeoServices** (ArcGIS REST API)
- **WMS** (Web Map Service)
- **WFS** (Web Feature Service)
- Data downloads in CSV, KML, GeoJSON, GeoTIFF, and PNG formats

### Wind and Solar Generation Data
Real-time and historical wind and solar generation data for the Balancing Authority area available at [transmission.bpa.gov](https://transmission.bpa.gov/business/operations/Wind/).

## Use Cases

- GIS data analysis and visualization for the Pacific Northwest energy grid
- Renewable energy monitoring (wind and solar generation)
- Transmission system capacity planning
- Energy research and data journalism
